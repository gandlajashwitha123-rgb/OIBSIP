# Iris Flower Classification

## Objective
Classify iris flowers using Machine Learning.

## Dataset Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Algorithms Used
- Random Forest Classifier

## Libraries
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Result
Achieved high accuracy in classifying iris flower species.
